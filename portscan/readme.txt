portscan для [TCP] портов

в ports_db происходит парсинг txt файла с iana.org, в котором прописано присвоение
имен портам (есть ссылка на локальную копию для более быстрого дебага)

в main происходит скан портов
в аргументы [необходимо] передать адрес
если не передать диапазон портов, будет использоваться 1-1024
[можно] передать число одновременных коннектов, таймаут соединения
есть опция verbose, при включении которой выводятся так же и закрытые порты
основная работа завязана на вызове connect_ex для разных портов.
